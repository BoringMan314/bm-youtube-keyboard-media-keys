隱私權說明（Chrome 擴充功能）

[B.M] YouTube 支援鍵盤媒體鍵 · bm-youtube-keyboard-media-keys

本擴充功能不蒐集、不上傳、不出售任何可識別個人身分的資料；未使用分析工具、廣告追蹤或遠端載入程式碼。

本機儲存

僅使用瀏覽器提供的 chrome.storage.local，於您的裝置本機儲存「是否啟用媒體鍵對應」等開關狀態，不會傳送到我們的伺服器（本專案亦未營運對應後端）。

權限與用途

擴充功能僅在必要時存取您已授權的 YouTube／YouTube Music 分頁，以回應快捷鍵並操作播放；除瀏覽器與 Google 服務本身之行為外，資料不會經由本擴充另傳他處。

聯絡與變更

若您對本說明有疑問，可透過 GitHub 儲存庫的 Issues 聯繫維護者。本頁內容如有更新，以儲存庫最新版本為準。

Privacy policy (Chrome extension)

[B.M] YouTube — Keyboard media keys · bm-youtube-keyboard-media-keys

This extension does not collect, upload, or sell personally identifiable information. It does not use analytics, ad trackers, or remotely loaded code.

Local storage

It uses chrome.storage.local only to save settings such as whether media-key mapping is enabled. Data stays on your device and is not sent to our servers (this project does not operate a backend for the extension).

Permissions

The extension accesses YouTube / YouTube Music tabs you already use, only to handle shortcuts and playback as described. It does not exfiltrate page data elsewhere.
